📖 Fable: The General's Scouts

将军派出五名侦察兵，每人一个方向同时出发。侦察兵们每隔一炷香放一只信鸽。将军把信鸽带来的情报标注在地图上。两名侦察兵一去不回——但将军根据已收到的三份情报做出了判断，打了胜仗。一个人看完五个方向要五天。五个人同时看只要一天。

The general dispatched five scouts, each heading in a different direction simultaneously. The scouts released carrier pigeons at regular intervals. The general plotted the intelligence on his map. Two scouts never returned — but the general made his judgment based on the three reports received and won the battle. One man scouting five directions takes five days. Five men scouting simultaneously takes only one day.

Send API 动态并行（数量不定时）

Send API Dynamic Parallelism (when the count is variable)

from langgraph.types import Send
class State(TypedDict):
    queries: list[str]
    results: Annotated[list[str], operator.add]  # Reducer 自动合并

def fan_out(state):  # 为每个 query 创建一个 Send
    return [Send("worker", {"query": q}) for q in state["queries"]]

def worker(state):  # 每个 worker 并行独立执行
    result = search(state["query"])
    return {"results": [result]}

builder.add_conditional_edges(START, fan_out, {"worker": "worker"})
