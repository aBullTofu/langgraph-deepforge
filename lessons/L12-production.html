my_agent/
├── agent/
│   ├── graph.py        # 图定义
│   ├── nodes.py        # 节点实现
│   ├── tools.py        # 工具定义
│   └── state.py        # State schema
├── tests/test_agent.py
├── langgraph.json      # Cloud 配置
├── pyproject.toml

langgraph.json

{
  "dependencies": ["."],
  "graphs": {"my-agent": "./agent/graph.py:graph"},
  "env": ".env"
}

CLI 命令

CLI 命令

langgraph dev       # 本地开发服务器（热重载）
langgraph build     # 构建 Docker 镜像
langgraph up        # 启动生产服务
langgraph deploy    # 部署到 LangGraph Cloud
