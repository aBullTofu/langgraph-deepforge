🔧 起点：你需要什么
🔧 Starting Point: What You Need

硬性前置条件（必须）
Hard Prerequisites (Required)

  • Python 3.10+ — 能写函数、类、装饰器、async/await
    Python 3.10+ — comfortable with functions, classes, decorators, async/await
  • 理解基本的 HTTP API 调用（requests / httpx）
    Basic understanding of HTTP API calls (requests / httpx)
  • 拥有至少一个 LLM API Key（DeepSeek / OpenAI / Anthropic 任一）
    Have at least one LLM API Key (DeepSeek / OpenAI / Anthropic)

软性前置（有帮助但不是必须）
Soft Prerequisites (Helpful but Not Required)

  • 了解 Transformer 的基本概念（Self-Attention 做了什么，不用推公式）
    Basic familiarity with Transformers (what Self-Attention does, no formulas required)
  • 用过至少一次 LangChain 的 ChatOpenAI 或类似 LLM 封装
    Used LangChain's ChatOpenAI or a similar LLM wrapper at least once
  • 听说过 RAG（检索增强生成）的概念
    Heard of RAG (Retrieval-Augmented Generation)
