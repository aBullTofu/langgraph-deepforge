🔥 六阶段深度学习引擎

🔥 Six-Phase Deep Learning Engine

Integrating Amanda Askell's fable pedagogy + MIT metacognition research + 50-minute deliberate practice + spaced reinforcement.

Phase 0

⚡ IGNITION · 认知点火

课前预热：一个故意设错的代码片段、一个无法用现有知识解决的问题。制造认知失调，激发学习欲望。

⚡ IGNITION · Cognitive Ignition

Pre-class warmup: a deliberately broken code snippet, a problem unsolvable with current knowledge. Creates cognitive dissonance to spark learning desire.

Phase 1

📖 FABLE → 🗺 GODVIEW

读寓言建立直觉 → 明确通关条件，跳过知识囤积。

📖 FABLE → 🗺 GODVIEW

Read a fable to build intuition → clarify victory conditions, skip knowledge hoarding.

Phase 2

⚔️ COMBAT · 三模式实战

AI 反派 / Debug 挑战 / 开放设计 — 三种角色轮换。

⚔ COMBAT · Triple-Mode Practice

AI antagonist / Debug challenge / Open design — rotate through three roles.

Phase 3

🔨 FORGE · 知识淬炼

3-2-1 复习法：3 句核心、2 个场景、1 个疑问。

🔨 FORGE · Knowledge Refinement

3-2-1 review method: 3 core insights, 2 application scenarios, 1 open question.

Phase 4

🔄 ECHO · 间隔回响

下次课开头默写上一课的 3 句话，再开始新课。

🔄 ECHO · Spaced Repetition

At the start of the next lesson, recall 3 key points from the previous one before beginning new material.
